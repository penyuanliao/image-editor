# Asset Editor

這是一個功能豐富的線上圖片編輯器，使用 Vue 3 和 TypeScript 建立。它提供了一個直觀的畫布界面，讓使用者可以輕鬆地新增和操作文字、圖片等元素，並提供縮放、對齊、裁切和匯出等功能。

## ✨ 功能特色

*   **畫布編輯**: 在可縮放和平移的畫布上新增、移動、旋轉和縮放物件。
*   **元素支援**: 支援文字和圖片元素，並可從本機上傳或貼上。
*   **多選與對齊**: 支援多選物件，並提供對齊工具。
*   **圖層管理**: (TBD) 未來可以新增圖層排序功能。
*   **鍵盤快捷鍵**: 支援刪除、移動、復原/重做等常用快捷鍵。
*   **圖片裁切**: 提供對單一圖片進行視覺化裁切的功能。
*   **匯出**: 將編輯後的畫布區域匯出為 PNG 圖片。

## 🛠️ 使用技術

*   **Vue 3**: 漸進式 JavaScript 框架。
*   **TypeScript**: 為 JavaScript 添加了靜態類型。
*   **Pinia**: Vue 的狀態管理庫。
*   **Sass/SCSS**: CSS 預處理器。

## 🚀 專案設定

1.  **安裝依賴**:
    ```bash
    npm install
    ```

2.  **啟動開發伺服器**:
    ```bash
    npm run dev
    ```

3.  **編譯和壓縮以進行生產**:
    ```bash
    npm run build
    ```

## 📁 專案結構

```
/src
├── assets          # 靜態資源，如圖片和字體
├── components      # 可重用的 Vue 元件
│   ├── Basic       # 基礎元件 (按鈕, 滾動條, etc.)
│   ├── EditorArea  # 主要編輯區相關元件
│   └── Panels      # 側邊面板 (素材, 屬性, etc.)
├── config          # 專案設定檔
├── store           # Pinia 狀態管理
├── styles          # 全域樣式和 SCSS 變數
├── types           # TypeScript 類型定義
└── Utilities       # 共用函式和工具
```
